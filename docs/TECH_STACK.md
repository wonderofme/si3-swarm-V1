# Tech Stack

## Runtime & Language

- **Node.js 22**
- **TypeScript 5.6.3** (ES2022, strict mode)

## Core Framework

- **ElizaOS** — multi-agent framework
- **Express 4.21** — REST API server

## Databases (dual support)

- **MongoDB 6.3.0**
- **PostgreSQL 8.13.0**
- Custom adapters for both

## AI/ML

- **OpenAI API** (GPT-4o-mini)
- **OpenAI Embeddings** (text-embedding-3-small)

## Integrations

- **Telegram Bot API** (Telegraf)
- **Nodemailer** (SMTP)
- **JWT authentication**

## Infrastructure

- **Docker** (multi-stage builds)
- **Akash Network** (decentralized cloud)
- **GitHub Actions** (CI/CD)

## Key Libraries

- **bcryptjs** (hashing)
- **sharp** (image processing)
- **zod** (validation)

## Architecture

- Plugin-based (onboarding, matching)
- Database adapter pattern
- Persistent cache layer
- REST API endpoints (`/api/chat`, `/api/metrics`, etc.)
